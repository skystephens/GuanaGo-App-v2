💳 ESTA SEMANA — Enviar links de pago a los 80 pax cotizados ▼
Las cotizaciones ya están en CotizacionesGG. PayU ya funciona. El paso siguiente es generar el link de pago para cada cotización y enviarlo por WhatsApp. Eso convierte una cotización en dinero real esta semana.

En Claude Code: "Lee la tabla CotizacionesGG de Airtable, filtra las que tienen Estado = 'Cotizada' o similar, genera el link de pago PayU para cada una y actualiza el campo con el link generado."
🤖 SEMANA 2 — Bot interno: agente orquestador en adminCerebro.tsx ▼
El endpoint POST /api/agentes/admin-chat ya está en la tarea de Tareas_To_do. Dale esta instrucción a Claude Code:

"En backend/routes/agentes.js, crear endpoint POST /api/agentes/admin-chat. El agente tiene acceso a: leer/crear/actualizar Tareas_To_do en Airtable, buscar en Procedimientos_RAG, guardar resumen en Leads.Memoria_Contexto del usuario con ID de Sky. Responde en español, es conciso, y lista qué acciones ejecutó al final de cada respuesta."
📌 SEMANA 2 — Stickers con QR + kit para primeros aliados ▼
Diseñar en Canva el sticker circular 5cm. QR apuntando a guanago.travel?utm_source=sticker. Cotizar en imprenta local: mínimo 100 unidades de cada versión. Llevar el kit cuando vayas a hablar con el organizador del torneo — si acepta el acuerdo, sus sedes reciben los stickers de "Evento oficial GuanaGO".
🌐 MCP en Claude Code — conectar Airtable directo ▼
Instalar el MCP de Airtable en Claude Code para que el bot pueda leer y escribir directamente en tus tablas desde VS Code. En la terminal de la carpeta del proyecto:

claude mcp add airtable

Pegar tu token PAT de Airtable cuando lo pida. Desde ese momento puedes decirle: "Lee las tareas críticas de hoy" y él las lee directamente sin que copies nada.
